## Abstract
This project implements a **serverless shopping website** using Amazon Web Services.  
The objective is to demonstrate cloud-native application development without managing servers while following **SDLC principles**.

---

## Objectives
- Understand serverless architecture
- Implement REST APIs using AWS
- Use NoSQL database for scalability
- Deploy a production-ready web application

---
